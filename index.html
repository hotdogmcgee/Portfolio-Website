My Portfolio

Music Book Reviews

A fullstack application for students, parents, and educators to find and review educational music books. The intention of the app is to use detailed input from educators to make clear what a given book contains and how it is or is not valuable. I have often dealt with the frustration of publisher-supplied information being the only source to use when assessing a book and wanted a way for everyone involved in music education to aggregate a knowledge base in a focused manner.

Client GitHub / Server GitHub / Live

Tonewood Data Project

A fullstack application that brings together luthiers worldwide to make it easy to find the properties of many tonewoods. API created for CRUD operations and log-in functionality.

Book Bestseller Finder

An app that allows the user to find lists of New York Times Bestsellers. The app will then allow a user to see if the book is available in the Harvard library system. This makes use of the New York Times Book API, as well as Harvard's LibCloud API.
